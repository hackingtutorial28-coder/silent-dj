## What is Silent DJ & Mixer?

**Silent DJ & Mixer** (also known as **Silent Wolf Audio**) is a professional‑grade, browser‑based DJ mixing workstation built by **Silent Wolf**. It runs on modern web technologies (React + Web Audio API) and is designed for both beginners and pro DJs — allowing you to mix tracks, apply effects, record sets, and broadcast to streaming services, all silently or with full output.

> Silent DJ & Mixer — professional mixing, zero latency, infinite creativity.

## ✨ Features

Silent DJ & Mixer comes packed with professional features:

- **Dual‑deck mixing** — seamless crossfader, gain, and EQ controls
- **Silent mode** — full mixing with speaker output disabled (headphone cue only)
- **Real‑time effects** — reverb, delay, flanger, phaser, bitcrusher, low‑pass filter
- **AI beat matching** — automatic BPM detection and sync between decks
- **Recording & playback** — capture your mix in high‑quality WAV/MP3
- **Stream integration** — direct output to Icecast, Shoutcast, or custom RTMP
- **Sample pads** — 8 programmable hot‑cue pads per deck
- **Visual wavef**orms — real‑time track preview with zoomable waveform
- **Playlist management** — drag & drop, folder scanning, ID3 tag reading
- **MIDI controller support** — map any MIDI device to faders, knobs, and buttons
- **Low latency** — Web Audio API with custom buffer management (<10ms)
- **BPM tap & nudge** — manual sync adjustment for imperfect tracks
- **Auto‑gain normalization** — consistent volume across tracks
- **Key detection** — Camelot wheel display for harmonic mixing

## ⚙️ Requirements

- **Modern web browser** (Chrome, Edge, Firefox, Safari) – Web Audio API required
- **Node.js** v18+ (only for the optional streaming server)
- **Audio interface** (optional, for headphone cue output)
