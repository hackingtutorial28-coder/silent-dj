## 💻 **Installation**

### 📦 **Prerequisites**

- Node.js 20.x or higher
- npm 10.x or higher  
- WhatsApp account
- Git

### 🔧 **Local Setup**

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/silent-dj.git
cd silent-dj

# 2. Install dependencies
npm install

# 3. Configure environment
nano settings.js
# Edit OWNER_NUMBER and MASTER_PASSWORD

# 4. Start the bot
npm start

# 5. Scan QR code with WhatsApp
