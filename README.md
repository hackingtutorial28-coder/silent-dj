## 🤖 24/7 HUNGRY

This bot isn't just awake, it's always on and always hungry for the next request. My system is designed for **round-the-clock availability**, utilizing health checks and auto-recovery to maintain a 99.9% uptime. Whether it's 3 AM or peak hours, the mix never stops.

- **🕒 Always-On Architecture**: Deployed on scalable cloud infrastructure (VPS/Railway) with auto-restart capabilities.
- **📈 Status Monitoring**: Real-time health checks ensure the bot recovers instantly from any crashes.
- **🌍 Global Low-Latency**: Optimized network routing guarantees snappy responses from anywhere in the world.

---

## ✨ Key Features

- **Dual-Deck Mixing**: Seamless crossfader, gain, and 3-band EQ.
- **Silent Cue Mode**: Full mixing via headphones only.
- **Audio Effects**: Reverb, delay, flanger, phaser.
- **Recording & Streaming**: Save mixes to WAV/MP3 or broadcast live.
- **Hardware Support**: MIDI controller mapping and keyboard shortcuts.
