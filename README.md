## ✨ Key Features

- **Dual-Deck Mixing**: Seamless crossfader, gain, and 3-band EQ.
- **Silent Cue Mode**: Full mixing via headphones only.
- **Audio Effects**: Reverb, delay, flanger, phaser.
- **Recording & Streaming**: Save mixes to WAV/MP3 or broadcast live.
- **Hardware Support**: MIDI controller mapping and keyboard shortcuts.
