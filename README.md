## 📌 What is Silent DJ & Mixer?

**Silent DJ & Mixer** is a professional, browser‑based DJ mixing workstation that runs 24/7. It lets you mix tracks, apply real‑time effects, record sets, and stream to online platforms — all with optional silent headphone cueing. Built with modern web technologies (React + Web Audio API) and designed for both beginners and pro DJs.

> Silent DJ & Mixer — always on, always hungry.

---

## 🤖 24/7 HUNGRY MODE

This bot never sleeps. It's always ready to mix, stream, and respond. Deployed on scalable cloud infrastructure with auto‑healing and global low‑latency routing.

- **🕒 Always‑On Architecture** – Auto‑restart on crash, health checks every 30s.
- **📈 Real‑time Monitoring** – Uptime dashboard, error logging, performance metrics.
- **🌍 Global Edge Network** – CDN delivery for static assets, WebSocket fallback.

## ✨ Full Feature List

### 🎚️ Mixing Engine
- Dual‑deck mixer with independent gain, 3‑band EQ, and crossfader
- Silent cue mode – headphones only, no speaker output
- Beat‑sync and BPM detection (AI‑powered)
- Tempo nudge and pitch shift
- Auto‑gain normalization

### 🎛️ Effects Suite
- Reverb, delay, flanger, phaser, chorus
- Distortion, bitcrusher, low‑pass/high‑pass filter
- Real‑time parameter control (knobs/faders)
- Chain up to 4 effects per deck

### 📀 Playlist & Library
- Drag‑and‑drop file loading (MP3, WAV, FLAC, OGG, AAC)
- ID3 tag reading (artist, title, BPM, key)
- Search, sort, and filter by BPM/key
- Create and save playlists (m3u/pls)

### 🎤 Recording & Streaming
- Record mixes to WAV/MP3 with metadata
- Auto‑split on silence
- Direct streaming to Icecast, Shoutcast, RTMP
- Now‑playing metadata updates

### 🎹 Hardware Support
- MIDI controller mapping (learn mode)
- Keyboard shortcuts (customizable)
- OSC support (optional)

### 👁️ Visuals
- Zoomable waveform with beat grid
- Real‑time spectrum analyzer
- Track progress and elapsed/remaining time
- Deck status indicators (playing, cue, sync)

### 🔧 Utilities
- BPM tap button
- Key detection (Camelot wheel)
- Track normalization (LUFS)
- Export mix to SoundCloud/YouTube (API integration)

### 🔐 Privacy & Control
- Local‑first mode – no cloud dependency
- Optional user accounts and session saving
- Self‑hosted or cloud deployment
