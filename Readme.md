## ✨ Features

- 🎚️ **Silent mode** – full mixing controls, zero speaker output (optional)
- 🎛️ **Real‑time audio mixer** – low‑latency faders, EQ, and effects
- 📀 **DJ‑grade cue system** – pre‑listen on headphones only
- ☁️ **Cloud sync** – save your sets and loops
- 🧠 **AI beat matching** & **BPM detection**
- 🔌 **Plugins support** – add your own effects or instruments

## 🚀 Quick Start

### Prerequisites

- Node.js 18+ and npm
- Audio interface (optional, for silent cueing)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/silent-dj-mixer.git

# Navigate to project
cd silent-dj-mixer

# Install dependencies
npm install

# Start the app
npm run dev
